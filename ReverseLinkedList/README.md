# Reverse Linked List

![Easy](https://img.shields.io/badge/Difficulty-Easy-green)

Given the `head` of a singly linked list, reverse the list, and return *the reversed list*.

**Example:**
```
Input: 1->2->3->4->5
Output: 5->4->3->2->1
```

**Constraints:**

- The number of nodes in the list is the range `[0, 5000]`.
- `-5000 <= Node.val <= 5000`
